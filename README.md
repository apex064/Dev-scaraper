# 🚀 DevScrape — Developer Website Scraper & Inspector

**DevScrape** is a powerful web-based tool built with Flask that allows developers to analyze and extract HTML, assets, metadata, and technologies from any public website.

🎯 Built for developers, tech reviewers, and data enthusiasts.

---

## 🌐 Live Demo

> 🔗 [https://dev-scaraper.onrender.com](https://dev-scaraper.onrender.com)

---

## 🔧 Features

| Mode              | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| 🧪 **Developer Mode** | Extracts full HTML, assets (CSS/JS/images), and packages into a downloadable ZIP |
| 🕵️ **Full Scraper Mode** | Extracts HTML, metadata, technology stack (frameworks, CMS, etc), emails, and social links |
